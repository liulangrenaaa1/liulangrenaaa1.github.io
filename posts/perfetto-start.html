第一篇：这个博客是怎么部署到 GitHub Pages 的

2026-04-16 · 建站实战

这篇记录本次博客从 0 到 1 的上线过程，目标是：快速可访问、风格统一、后续维护成本低。

1. 创建仓库

  • 创建仓库：liulangrenaaa1.github.io
  • 用户站点规则：仓库名必须是 <username>.github.io

2. 提交博客结构

  • index.html：首页
  • assets/style.css：样式
  • posts/*.html：文章页

3. 关键兼容项

  • 添加 .nojekyll，避免默认 Jekyll 干预静态资源路径
  • 确保链接都使用相对路径，便于后续维护

4. 上线与验证

  • 站点地址：https://liulangrenaaa1.github.io
  • 仓库地址：https://github.com/liulangrenaaa1/liulangrenaaa1.github.io
  • 首次发布可能有几分钟缓存/构建延迟，属于正常现象

5. 下一步计划

  1. 接入评论系统（giscus）
  2. 增加标签页与归档页
  3. 补充性能分析系列文章（Perfetto / Systrace）

如果你也想快速搭一个工程感博客，这套方案是低成本且稳定的起点。
